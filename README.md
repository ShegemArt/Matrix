# Решение систем линейных уравнений (C++)

## Обзор

Данный проект представляет собой реализацию небольшой библиотеки линейной алгебры на языке C++ и консольного приложения для решения систем линейных уравнений вида:

[
Ax = b
]

где:

* `A` — квадратная матрица размера (n × n),
* `b` — вектор размера n,
* `x` — искомый вектор решения.

Программа позволяет пользователю выбирать различные численные методы решения и анализировать их поведение.

---

## Структура проекта

Проект состоит из следующих основных компонентов:

### 1. Структура Matrix (`Matrix`)

Представляет квадратную матрицу.

**Поля:**

* `Dimension` — размер матрицы (n × n)
* `numbersMatrix` — двумерный контейнер с элементами матрицы

**Основные методы:**

* `InputFromConsole()` — ввод матрицы с консоли
* `Print()` — вывод матрицы
* `MakeIdentityMatrix(n)` — создание единичной матрицы
* `Transpose()` — транспонирование матрицы
* `SubMatrix()` — создание подматрицы

**Поддерживаемые операции:**

* Сложение и вычитание матриц
* Умножение матриц
* Умножение на скаляр
* Умножение матрицы на вектор

---

### 2. Структура Vector (`Vector`)

Представляет математический вектор.

**Поля:**

* `Dimension` — размерность вектора
* `numbersVector` — контейнер с элементами вектора

**Основные методы:**

* `InputFromConsole()` — ввод вектора
* `Print()` — вывод вектора
* `MakeZeroVector(n)` — создание нулевого вектора

**Поддерживаемые операции:**

* Сложение и вычитание векторов
* Умножение на скаляр
* Внешнее произведение (вектор × вектор → матрица)

---

### 3. Функции норм

Используются для оценки сходимости и точности.

**Нормы матриц:**

* `L1MatrixNorm` — максимальная сумма по столбцам
* `InfMatrixNorm` — максимальная сумма по строкам

**Нормы векторов:**

* `L1VectorNorm` — сумма модулей элементов
* `InfVectorNorm` — максимальный по модулю элемент
* `L2VectorNorm` — евклидова норма

## Реализованные методы решения СЛАУ

### 1. Метод простых итераций (MPI)

* Итерационный метод
* Использует преобразование:
  [
  x = Bx + c
  ]
* Автоматически преобразует систему при отсутствии сходимости
* Критерии остановки:

  * оценка погрешности (если ||B|| < 1)
  * невязка в противном случае

---

### 2. Метод Зейделя (Seidel)

* Итерационный метод
* Использует обновлённые значения в процессе вычислений
* Проверяет диагональное преобладание
* При необходимости преобразует систему
* Критерий остановки — невязка

---

### 3. PLU-разложение

* Прямой метод
* Представляет матрицу в виде:
  [
  PA = LU
  ]
* Использует частичный выбор главного элемента
* Решение выполняется в два этапа:

  1. Прямой ход (Ly = b)
  2. Обратный ход (Ux = y)

---

### 4. QR-разложение (метод Хаусхолдера)

* Прямой метод
* Представляет матрицу в виде:
  [
  A = QR
  ]
* Использует отражения Хаусхолдера
* Решение системы:
  [
  Rx = Q^T b
  ]

---

## Основная программа

Программа реализует консольный интерфейс, в котором пользователь может:

1. Выбрать метод:

   * 1 — MPI
   * 2 — Зейдель
   * 3 — PLU
   * 4 — QR
2. Ввести:

   * размер матрицы `n`
   * матрицу `A`
   * вектор `b`
   * точность `epsilon`
3. Получить:

   * вектор решения `x`
   * количество итераций (для итерационных методов)

Программа работает в цикле до выбора выхода (`0`).

## Ограничения

* Поддерживаются только квадратные матрицы
* Нет полной проверки вырожденности матриц
* Численная устойчивость зависит от входных данных
